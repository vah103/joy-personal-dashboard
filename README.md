# Joy Personal Dashboard

A responsive personal dashboard prototype for Vanh, covering priority email, sales viewings, active projects, and daily tasks.

## Sample behavior

- Add and complete daily tasks.
- Add or archive projects.
- Mark the priority email as done.
- Confirm or return a sample viewing to pending.
- Keep changes in the current browser using `localStorage`.

This version uses sample data only. It does not connect to a real email account or CRM yet.
